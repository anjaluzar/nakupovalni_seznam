Kratek opis projekta

Aplikacija je sestavljena iz React frontend-a in Node/Express backend-a, ki uporablja MongoDB (Mongoose) za shranjevanje nakupovalnih seznamov. Frontend komunicira z REST API-jem na poti /api/lists za CRUD operacije.
